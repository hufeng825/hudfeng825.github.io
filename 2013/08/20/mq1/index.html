MQ 笔记之 mac 下安装环境

下载

从下载地址中下载包
解压包然后把包放到一个自己不会手一抖就删掉的位置，此处我放到了本地的document下
如下图

添加环境到path

把/你的保存路径/apache-activemq-5.8.0/bin/activemq 添加到 PATH 中

  1. cd
  2. 执行 vi .profile
  3. 输入 i
  4. 插入 export MQ=/你的路径/apache-activemq-5.8.0/bin/macosx/activemq
  5. wq
  6. source .profile

启动mq

$Mq start
输入账号 admin 密码 admin

注意 如果用代理上网 请把网络环境设置下 如下图
